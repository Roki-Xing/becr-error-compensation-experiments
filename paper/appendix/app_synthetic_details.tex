The corrected synthetic suite contains four evidence blocks:
\begin{enumerate}[leftmargin=1.5em]
\item theorem-regime $2$D validation with explicit theorem-condition logging;
\item high-dimensional fixed-stale diagnostics with flat and anisotropic
spectra;
\item refresh sweeps over $K\in\{1,10,50,200,\infty\}$ with explicit
moving-projection state modes;
\item anisotropic noisy projection diagnostics with common random numbers.
\end{enumerate}

Theorem-regime values are reproduced in Table~\ref{tab:theorem-values}. The
high-dimensional summary shows that clipping and \BECR\ drive the orthogonal
gradient to numerical zero while leaving a nonzero parallel floor. The refresh
sweep shows that frequent refresh reduces stale failure and that corrected state
transport changes behavior materially when the subspace moves. The anisotropic
setting records paired differences and claim text directly in the aggregate
summary to prevent mismatch between booleans and prose.
